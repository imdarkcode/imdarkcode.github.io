# IMDARKCODE

Portfólio pessoal desenvolvido com **HTML, CSS e JavaScript** puro, seguindo a metodologia **BEM** para organização e nomenclatura das classes CSS, com arquivos separados por componente.

## 🎨 Design

O design foi prototipado no **Figma** e se baseia na estética **brutalista** — fundo preto, tipografia branca em peso bold e detalhes discretos. O projeto conta com **design responsivo** para diferentes tamanhos de tela.

## 🛠️ Tecnologias

- HTML, CSS e JavaScript puro
- Fonte **Satoshi**
- Ícones **Iconoir**
- Animações **Animate.css**
- Prototipação **Figma**
