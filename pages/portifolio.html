IMDARKCODE

  • Categoria

    Portifólio

  • Ano

    2026

  • Tecnologias

    HTML, CSS, JavaScript

IMDARKCODE é um portfólio pessoal multi-página construído com HTML, CSS e JavaScript puro, seguindo a metodologia BEM para organização e nomenclatura das classes CSS, com arquivos separados por componente.

O design foi prototipado no Figma e se baseia na estética brutalista — fundo preto, tipografia branca em peso bold e detalhes discretos. O projeto utiliza a fonte Satoshi, ícones do Iconoir, animações da biblioteca Animate.css e conta com design responsivo para diferentes tamanhos de tela.

A navegação é feita por um menu overlay que cobre toda a tela ao ser acionado, e as páginas que compõem o site são: Página Inicial, Meus Projetos, Detalhes do Projeto, Sobre Mim e Contato.
